### 🎵 Media Control, The Sovereign Way

These shortcuts talk to **Beatify**, which talks to my **MPD** (Groove Basin):

- `Ctrl+Shift+N`: Next track (skips song + reduces its rating by 0.5)
- `Ctrl+Alt+0` through `Ctrl+Alt+9`: Rate current track from 5★ to 0.5★
- `Ctrl+Alt+1`: Flag for deletion
- `Ctrl+Alt+2`: Flag for technical issue → re-download fresh copy

My **auto-DJ** plays only rated 5+ tracks (proportional to rating weight).

### 🖥️ Window Placement Shortcuts

Inspired by tiling WMs but allows overlap:

* `Alt+Shift+Q`: Top-left 720p size
* `Alt+Shift+W`: Top-right 720p size
* `Alt+Shift+A`: Left 1080p half
* `Alt+Shift+S`: Right 1080p half
* `Alt+Shift+Z`: Centered window with padding
* `Alt+Shift+X`: Move to next screen
* `Alt+Shift+C`: Cascade windows
* `Alt+Shift+V`: Maximize/restore with padding

> Tiling managers are great — but I like my app windows to float just right.
