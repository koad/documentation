### going in circles
```bash

```

### walking
```bash
BUSY_CURSOR='⠁⠃⠇⡆⡄⡀⠀⠈⠘⠸⢰⢠⢀' # longer strides
BUSY_CURSOR='⠁⠃⠆⡄⡀⠈⠘⠰⢠⢀' # left right left

BUSY_CURSOR='⡇⢇⢣⢱⢸⡸⡜⡎'   # back'n forth
```
