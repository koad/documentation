# Meteor JS Cheatsheet `src/.meteor/packages`

###  A basic app
```
koad:io-core
```
###  with routing
```
koad:io-core
koad:io-router
```
###  with authenicated routing
```
koad:io-core
koad:io-router
koad:io-accounts-core
koad:io-accounts-ui
koad:io-meteor-helpers
```
###  with front end UI
```
koad:io-core
koad:io-router
koad:io-accounts-core
koad:io-accounts-ui
koad:io-template-helpers
koad:io-meteor-helpers
koad:io-style-basic-bitch
```
###  without authenicated routing
```
koad:io-core
koad:io-router
koad:io-template-helpers
koad:io-style-basic-bitch
```
